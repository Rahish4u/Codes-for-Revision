Differences Between HTML and HTML5

HTML HTML5
Primarily a static language with limited multimedia capabilities. Supports multimedia elements like <video> and <audio> natively.
Uses <div> tags extensively for layout. Introduces semantic elements like <header>, <footer>, <nav>, etc., for better content structure.
Relies on external plugins (e.g., Flash) for media playback. Does not require external plugins for media playback, offering native support.
